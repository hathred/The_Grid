  - folder: 000_Project_Manifests
    purpose: Core metadata, project manifest files, and overall configuration.
  - folder: 010_Data_RDBMS_NoSQL
    purpose: Database installers and data system tools (e.g., PostgreSQL, MongoDB).
  - folder: 020_SCM_Version_Control
    purpose: Source Code Management and version control archives (e.g., Git).
  - folder: 030_Auth_Identity_Management
    purpose: Identity Provider and authentication tools and archives.
  - folder: 040_Web_Server_Frontend
    purpose: Web server installers and core frontend files (e.g., Apache, Nginx).
  - folder: 050_Network_Core_Services
    purpose: Archives for core networking infrastructure (e.g., Outranet services).
  - folder: 060_Virtualization_Containerization
    purpose: Container and virtualization engine installers (e.g., Docker, Kubernetes/Orchestrator).
  - folder: 070_DevOps_CI_CD
    purpose: Continuous Integration/Continuous Deployment tools (e.g., Jenkins).
  - folder: 080_Observability_ELK_Stack
    purpose: Monitoring, logging, and metrics systems (e.g., Monitoring tools).
  - folder: 090_Messaging_IoT_Broker
    purpose: Message queue systems and IoT broker archives (e.g., MQTT).
  - folder: 100_Dependency_Runtime_Archives
    purpose: Stored runtime and dependency archives for various languages (e.g., Java, Node, Python pip).
  - folder: 105_System_Automation_Scripts
    purpose: OS-level batch files and system automation scripts.
  - folder: 195_Documentation_Knowledge_Base
    purpose: Project documentation, technical wikis, and knowledge reference materials.
  - folder: 200_Content_Data_Source
    purpose: General content archives, raw data, and content creation resources (e.g., Religion, Role play).
  - folder: 210_Project_Dev_Tools
    purpose: Specific project development and utility tools (e.g., Grok, The Generator).
  - folder: 990_Project_Automation
    purpose: Project-level utility scripts (formerly top-level scripts).
  - folder: 995_Project_Logs
    purpose: Logs and diagnostic outputs from the sorting process and general system operations.
