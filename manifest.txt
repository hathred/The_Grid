Included Software

### Design and Graphics
- **Adobe**: Creative software suite (e.g., Photoshop, Illustrator). [Adobe](https://www.adobe.com/)
- **GIMP**: Open-source image editor, alternative to Photoshop. [GIMP](https://www.gimp.org/)
- **Gravit Designer**: Vector graphics design tool, often used for UI design. [Gravit Designer](https://www.designer.io/en/)
- **Krita**: Digital painting and illustration software. [Krita](https://krita.org/en/)
- **Paint.NET**: Free image editing software for Windows with an intuitive interface. [Paint.NET](https://www.getpaint.net/)

### Development Tools
- **Apache NetBeans**: IDE that supports multiple programming languages. [NetBeans](https://netbeans.apache.org/)
- **Atom**: Free and open-source text editor designed for code. [Atom](https://atom.io/)
- **Docker**: Platform for developing, shipping, and running applications in containers. [Docker](https://www.docker.com/)
- **Eclipse**: Integrated development environment (IDE) for Java. [Eclipse](https://www.eclipse.org/)
- **Ruby on Rails**: Web application framework for Ruby. [Ruby on Rails](https://rubyonrails.org/)
- **Visual Studio Code**: Extensible code editor with numerous extensions. [VS Code](https://code.visualstudio.com/)

### Media Handling
- **Audacity**: Open-source audio editing software. [Audacity](https://www.audacityteam.org/)
- **ffmpeg**: Command-line tool for multimedia processing. [ffmpeg](https://ffmpeg.org/)
- **HandBrake**: Open-source video transcoder for files and DVDs. [HandBrake](https://handbrake.fr/)
- **VLC Media Player**: Versatile media player supporting various formats. [VLC](https://www.videolan.org/vlc/)

### Networking
- **FileZilla**: Open-source FTP client with support for FTPS and SFTP. [FileZilla](https://filezilla-project.org/)
- **WireShark**: Network protocol analyzer for troubleshooting. [WireShark](https://www.wireshark.org/)

### Communication
- **Discord**: Voice, video, and text chat service; open-source variants available. [Discord](https://discord.com/)
- **Rocket.Chat**: Open-source team communication platform. [Rocket.Chat](https://rocket.chat/)

### System Utilities
- **7-Zip**: File archiver with a high compression ratio. [7-Zip](https://www.7-zip.org/)
- **Clonezilla**: Disk cloning and imaging tool. [Clonezilla](https://clonezilla.org/)
- **GParted**: Partition editor for enabling/disposing of disk partitions. [GParted](https://gparted.org/)
- **TImagePrinter**: Image printer solution for generating custom images. [TImagePrinter](http://www.biswajeet.com/)

### Others
- **Arduino**: Open-source electronics platform for building interactive projects. [Arduino](https://www.arduino.cc/)
- **BLHeliSuite**: Software for configuring BLHeli ESCs. [BLHeli](http://www.blhelisuite.com/)
- **Chromium**: Open-source web browser that forms the basis for Google Chrome. [Chromium](https://www.chromium.org/)
- **Fusion 360**: CAD, CAM, and CAE tool for product design. [Fusion 360](https://www.autodesk.com/products/fusion-360/overview)
- **Inkscape**: Open-source vector graphics editor. [Inkscape](https://inkscape.org/)
- **Meshmixer**: Tool for 3D modeling and printing. [Meshmixer](http://www.meshmixer.com/)
- **PicEngrave**: Software for engraving images. [PicEngrave](https://www.picengrave.com/)
- **Selenium**: Automated testing tool for web applications. [Selenium](https://www.selenium.dev/)
- **Simplify3D**: Comprehensive 3D printing software solution. [Simplify3D](https://www.simplify3d.com/)
- **Thunderbird**: Open-source email client from Mozilla. [Thunderbird](https://www.thunderbird.net/)
- **VirtualBox**: Open-source virtualization software. [VirtualBox](https://www.virtualbox.org/)
- **VMware**: Virtualization software for running multiple operating systems. [VMware](https://www.vmware.com/)
